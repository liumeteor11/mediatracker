# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-28

### Added
- Initial release of MediaTracker AI.
- Electron-based desktop application.
- AI-powered media tracking and recommendations using Moonshot AI, OpenAI, DeepSeek, etc.
- Web search integration (Google, Bing, Yandex).
- Secure API key management with local encryption.
- Media collection management (Movies, Books, TV Series, etc.).
- Dashboard with statistics and charts.
